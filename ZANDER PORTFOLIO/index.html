VortexShift Cipher

A Hybrid Cryptographic Cipher — Concept & Technical Reference

Combining: Positional Caesar Shift · Rail Fence Transposition · Mirror Fold

Created by: Zander Rhoi Villanueva, Marcus Aurelius Rivera, Mark Anthony Salario, Lawrence Francisco

Abstract

This paper presents the VortexShift Cipher, a novel hybrid encryption scheme developed as an original contribution to the study of classical cryptographic techniques. The cipher draws from three foundational concepts — Positional Caesar Shift, Rail Fence Transposition, and Mirror Fold — and integrates them into a single multi-layer algorithm that provides significantly higher resistance to cryptanalysis than any of its component techniques applied individually.

Part I — Component Cipher Foundations

Before presenting the VortexShift Cipher itself, it is essential to understand the three classical ciphers that form its building blocks.

1.1 Positional Caesar Shift
The Positional Caesar Shift is an extension of the classical Caesar Cipher in which the shift applied to each character varies based on the character's position in the message. For a given key k, the shift at position i is defined as (k + 2i), meaning that each successive character is shifted two more positions than the one before it. This positional dependency makes the cipher significantly more resistant to frequency analysis.

ENCRYPTION: shift(i) = (key + 2 * i) mod 26
            C[i] = (P[i] - base + shift(i)) mod 26 + base

DECRYPTION: shift(i) = (key + 2 * i) mod 26
            P[i] = (C[i] - base - shift(i)) mod 26 + base

1.2 Rail Fence Transposition
The Rail Fence Cipher is a form of transposition cipher in which the plaintext characters are written diagonally downward and upward across a set of imaginary horizontal rails, creating a zigzag pattern, then read row-by-row. Crucially, no character's value is changed — only their order is rearranged.

1.3 Mirror Fold
The Mirror Fold is a transposition technique in which the content of each rail — as produced by the Rail Fence pattern — is individually reversed before being concatenated into the final output. This creates an asymmetric scrambling effect: positions near the center of the message are more displaced than those near the edges.

ENCRYPTION:
1. Apply Rail Fence pattern to obtain R rail segments
2. Reverse each rail segment independently: rail[r] = reverse(rail[r])
3. Concatenate reversed rails: C = rail[0] + rail[1] + ...
Part II — The VortexShift Cipher

2.1 Concept and Inspiration
The VortexShift Cipher was designed with a single guiding principle: no single layer of classical cryptography is robust enough on its own, but layers applied in the right order create compounding security. The name "VortexShift" reflects how a message spirals through three transformations — Positional Caesar, Rail Fence, and Mirror Fold — before emerging as unrecognizable ciphertext.

ENCRYPTION: C = MirrorFold( RailFence( PosShift(P, key), rails ), rails )
DECRYPTION: P = PosShift-1( RailFence-1( MirrorFold-1(C, rails), rails ), key )

VORTEXSHIFT CIPHER

A Hybrid Cryptographic Cipher

  • Layer 2: Rail Fence Transposition (Permutation)

OVERVIEW

What is the VortexShift Cipher?

VortexShift is an original hybrid encryption algorithm that chains three classical cryptographic techniques into a single, multi-layer cipher. Each layer attacks a different vulnerability — character frequency, character order, and structural position.

LAYER 1: POSITIONAL CAESAR

Variable Shift Substitution

Definition: Each character's shift grows with its position. Shift at index i = key + (2 × i). Eliminates uniform frequency patterns.
Example: HELLO (key = 3) → KJSUZ

LAYER 2: RAIL FENCE

Zigzag Transposition

Definition: Characters are written in a zigzag pattern across R rails, then read rail-by-rail. Reorders characters without changing values.
Example: KJSUZ (rails = 3) → KSZJU

LAYER 3: MIRROR FOLD

Segment Reversal

Definition: Reverses each rail segment independently. Creates asymmetric scrambling near message center.
Example: KSZJU (rails = 3) → ZSK · UJ → ZSKUJ

DESIGN PHILOSOPHY

Compounding Security

The spiral nature of the algorithm rotates characters through substitution (Layer 1), transposition (Layer 2), and structural folding (Layer 3), forcing cryptanalysts to break all layers in reverse order.

# ══════════════════════════════════════════════════════════════
#  LAYER 1 — POSITIONAL CAESAR SHIFT
# ══════════════════════════════════════════════════════════════
def positional_caesar_encrypt(text, key):
    result = []
    for i, ch in enumerate(text):
        if ch.isalpha():
            shift = (key + 2 * i) % 26
            base  = ord('A') if ch.isupper() else ord('a')
            result.append(chr((ord(ch) - base + shift) % 26 + base))
        else:
            result.append(ch)
    return "".join(result)

def positional_caesar_decrypt(text, key):
    result = []
    for i, ch in enumerate(text):
        if ch.isalpha():
            shift = (key + 2 * i) % 26
            base  = ord('A') if ch.isupper() else ord('a')
            result.append(chr((ord(ch) - base - shift) % 26 + base))
        else:
            result.append(ch)
    return "".join(result)


# ══════════════════════════════════════════════════════════════
#  LAYER 2 — RAIL FENCE TRANSPOSITION
# ══════════════════════════════════════════════════════════════
def rail_fence_encrypt(text, rails=3):
    if rails < 2 or rails >= len(text):
        return text

    fence = [[] for _ in range(rails)]
    rail, direction = 0, 1

    for ch in text:
        fence[rail].append(ch)
        if rail == 0:
            direction = 1
        elif rail == rails - 1:
            direction = -1
        rail += direction

    return "".join("".join(r) for r in fence), fence

def rail_fence_decrypt(cipher, rails=3):
    n = len(cipher)
    if rails < 2 or rails >= n:
        return cipher

    # Build pattern
    pattern = []
    rail, direction = 0, 1
    for _ in range(n):
        pattern.append(rail)
        if rail == 0:
            direction = 1
        elif rail == rails - 1:
            direction = -1
        rail += direction

    # Count chars per rail
    counts = [pattern.count(r) for r in range(rails)]
    # Slice cipher into rails
    rails_data = []
    idx = 0
    for c in counts:
        rails_data.append(list(cipher[idx:idx+c]))
        idx += c

    # Reconstruct
    result = []
    pointers = [0] * rails
    for r in pattern:
        result.append(rails_data[r][pointers[r]])
        pointers[r] += 1
    return "".join(result)


# ══════════════════════════════════════════════════════════════
#  LAYER 3 — MIRROR FOLD
# ══════════════════════════════════════════════════════════════
def mirror_fold_encrypt(text, rails=3):
    """Split text by rail-fence pattern, then reverse each rail segment."""
    n = len(text)
    if rails < 2 or rails >= n:
        return text[::-1], [text[::-1]]

    pattern = []
    rail, direction = 0, 1
    for _ in range(n):
        pattern.append(rail)
        if rail == 0:
            direction = 1
        elif rail == rails - 1:
            direction = -1
        rail += direction

    fence = [[] for _ in range(rails)]
    for i, ch in enumerate(text):
        fence[pattern[i]].append(ch)

    # Reverse each rail
    mirrored = [list(reversed(r)) for r in fence]
    return "".join("".join(r) for r in mirrored), mirrored

def mirror_fold_decrypt(cipher, rails=3):
    """Reverse of mirror fold: split back, reverse each segment, re-weave."""
    n = len(cipher)
    if rails < 2 or rails >= n:
        return cipher[::-1]

    pattern = []
    rail, direction = 0, 1
    for _ in range(n):
        pattern.append(rail)
        if rail == 0:
            direction = 1
        elif rail == rails - 1:
            direction = -1
        rail += direction

    counts = [pattern.count(r) for r in range(rails)]
    rails_data = []
    idx = 0
    for c in counts:
        seg = list(cipher[idx:idx+c])
        rails_data.append(list(reversed(seg)))   # un-reverse
        idx += c

    result = []
    pointers = [0] * rails
    for r in pattern:
        result.append(rails_data[r][pointers[r]])
        pointers[r] += 1
    return "".join(result)


# ══════════════════════════════════════════════════════════════
#  FULL VORTEXSHIFT — ENCRYPT + DECRYPT
# ══════════════════════════════════════════════════════════════
def vortexshift_encrypt(plaintext, key, rails=3):
    after_l1 = positional_caesar_encrypt(plaintext, key)
    after_l2_text, fence_data = rail_fence_encrypt(after_l1, rails)
    after_l3_text, mirror_data = mirror_fold_encrypt(after_l2_text, rails)
    return after_l1, after_l2_text, after_l3_text, fence_data, mirror_data

def vortexshift_decrypt(ciphertext, key, rails=3):
    after_u3 = mirror_fold_decrypt(ciphertext, rails)
    after_u2 = rail_fence_decrypt(after_u3, rails)
    after_u1 = positional_caesar_decrypt(after_u2, key)
    return after_u3, after_u2, after_u1

    print(f"    {GRAY}Formula: original = (char - shift) mod 26{RESET}\n")

    print(f"    {'Pos':<5} {'Char':<6} {'Shift':<8} {'Result':<8}")
    divider("·", GRAY, 40)

    for i, ch in enumerate(after_u2):
        if not ch.isalpha():
            print(f"    {i:<5} {GRAY}{ch!r:<6}{'—':<8}{'(skip)':<8}{RESET}")
            continue
        shift = (key + 2 * i) % 26
        base  = ord('A') if ch.isupper() else ord('a')
        res   = chr((ord(ch) - base - shift) % 26 + base)
        print(f"    {CYAN}{i:<5}{RESET} {YELLOW}{ch:<6}{RESET}{GREEN}{shift:<8}{RESET}{LIME}{res:<8}{RESET}")

    # ── RESULT ──
    divider("═", TEAL, 62)
    print(f"\n  {GREEN}{BOLD}✔  DECRYPTED RESULT{RESET}")
    print(f"\n  {GRAY}Ciphertext:{RESET}  {CYAN}{BOLD}{ciphertext}{RESET}")
    print(f"  {GRAY}Key       :{RESET}  {ORANGE}{key}{RESET}")
    print(f"  {GRAY}Rails     :{RESET}  {ORANGE}{rails}{RESET}")
    print(f"  {GRAY}Plaintext :{RESET}  {GREEN}{BOLD}{after_u1}{RESET}")
    divider("═", TEAL, 62)

    return after_u1

def get_key():
    while True:
        try:
            k = int(input(f"  {YELLOW}Enter key (integer): {RESET}").strip())
            return k
        except ValueError:
            print(f"  {RED}Key must be an integer. Try again.{RESET}")

def get_rails():
    while True:
        try:
            r = int(input(f"  {YELLOW}Enter number of rails [2-6, default 3]: {RESET}").strip() or "3")
            if 2 <= r <= 6:
                return r
            print(f"  {RED}Rails must be between 2 and 6.{RESET}")
        except ValueError:
            print(f"  {RED}Must be an integer. Try again.{RESET}")
